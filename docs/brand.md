# Tuji Brand

Logo files:
- `public/logo.svg` — full logo (hero/header use)
- `public/logo-icon.svg` — compact round icon for nav and favicon

Colors:
- Primary: #9da50eff (teal)
- Accent: #5f0925ff (dark slate)

Usage:
- Use `logo.svg` for hero and marketing headers.
- Use `logo-icon.svg` for compact places (nav, favicon).

Export suggestions:
- Provide PNG at 2x and 3x sizes (e.g., 2000px wide) for marketing.
- Keep favicon as 120x120 PNG or ICO derived from .`logo-icon.svg`
